## Trade Platform

### Purpose
This project is meant to emulate a trading platform.

### Technologies
- Java 17
- Spring Boot 
- Quarkus
- PostgresSQL
- MongoDB
- Redis

### Modules

- Onboarding
- Rates
- Trades
- Transfers
- Rate Simulator

## How to initialize Database
Setup a local database using the following command

```bash
docker-compose up -d db
```
